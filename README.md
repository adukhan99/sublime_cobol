# Sublime COBOL Syntax Highlighting

This repository provides COBOL syntax highlighting for Sublime Text. It includes support for various COBOL dialects and related mainframe languages.

## Supported Languages

The following syntax highlighting files are included:

- ACUCOBOL-GT
- BMS
- COBOL
- Enterprise COBOL
- GnuCOBOL (formerly OpenCOBOL)
- HLASM (IBM High Level Assembler)
- JCL (Job Control Language)
- MFU (Micro Focus Universe)
- PL/I
- Utreport

## Converter

This repository also includes a Python script, `converter.py`, to convert `.tmLanguage.json` files to the `.sublime-syntax` format used by Sublime Text 3 and later.

### Usage

```shell
python converter.py <input_file.json> <output_directory>
```
